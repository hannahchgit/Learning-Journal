# 📓 Learning Journal

An interactive, browser-based journal for students to reflect on and consolidate what they've learned — no installation required.

## Features

- **4 tabbed sections** — Reflect, Capture, Apply, Review
- **Reflect** — Topic, pre-session mood check, biggest takeaway, and surprises
- **Capture** — Key concepts & notes, questions that arose, resources to explore
- **Apply** — Real-world connections, action steps, and a "teach it back" prompt
- **Review** — Session star rating, what to do differently, confidence level, and next steps
- **Multiple entries** — Save and switch between journal entries; stored locally in the browser
- **Progress bar** — Fills as you complete each section
- **Print-ready** — Clean print layout with no buttons or UI clutter
- **Word count** on key fields

## How to Use

1. Open `index.html` in any modern web browser
2. Fill in the **Topic** field and choose a mood
3. Work through each tab — Reflect → Capture → Apply → Review
4. Click **Save** to store the entry locally
5. Click **+ New Entry** to start a fresh journal page
6. Click **Print** for a clean physical copy

## Live Demo

Enable GitHub Pages (Settings → Pages → Deploy from `main`) to share a live link with students.
